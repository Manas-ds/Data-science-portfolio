# Gradio Web App

A simple UI to play with the social listening engine without touching code.

## What it looks like
Two tabs:
- **Search** — query posts, filter by intent/sentiment, see trends
- **Trending Topics** — see what's rising right now

## 🚀 Launch the app
```bash
python app/gradio_app.py

 ## How to use
    Type a query (like "delivery refund")
    Pick intent/sentiment filters (or leave as "Any")
    Hit Run
    See:
        Summary bullets
        Top themes
        Evidence table
        Trend chart
    
  Download results as JSON if you want
