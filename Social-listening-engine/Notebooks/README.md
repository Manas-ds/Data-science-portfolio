# Notebooks

This folder contains the main Jupyter notebook for the project.

## 📒 Files
- `social_listening_engine.ipynb` — The full pipeline from start to finish

## 🧠 What's in the notebook
1. Load data (tweet_eval + synthetic timestamps)
2. Clean text
3. Classify intent (rules + zero-shot)
4. Build FAISS search index
5. Discover topics via clustering
6. Find trending topics
7. Launch Gradio dashboard

## ▶️ How to run

jupyter notebook social_listening_engine.ipynb
